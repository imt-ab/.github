# Security Policy

If you believe you have identified a security issue in material published by IMT AB, please do not disclose it publicly before contacting us.

## Reporting a vulnerability

Please report security-related issues through our website contact channel:

[https://imtab.net](https://imtab.net/en/kontakt/)

When possible, include:

- a clear description of the issue
- affected repository, file, or component
- steps to reproduce
- expected and actual behavior
- potential impact

## Disclosure approach

We review security reports as they are received and assess them based on severity, impact, and reproducibility.

Please avoid public disclosure until the issue has been reviewed and appropriate action has been taken.
